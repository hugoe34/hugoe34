## 🚀 Features

- 🎵 Stream music directly from Deezer
- 📂 Browse and manage playlists
- ⏯️ Play, pause, skip tracks
- 🔍 Search for songs, albums, or artists
- ⚡ Smooth real-time audio playback
- 💻 Built with C# and .NET for Windows desktop
